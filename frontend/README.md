# LedgerLink Frontend

This is the frontend application for LedgerLink - an account reconciliation system with enhanced account linking features.

## Features

- Modern UI with responsive design
- Account reconciliation between AR and AP ledgers
- Support for various data sources including Xero and CSV uploads
- Interactive results dashboard
- Account linking capabilities

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Deployment

This frontend is deployed on Vercel and should automatically update when changes are pushed to the main branch.
